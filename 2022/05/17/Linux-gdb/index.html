step 和 next

run全速执行。

starti执行第一条机器指令。

next 单步到程序源代码的下一行，不进入函数。
nexti 单步一条机器指令，不进入函数。
step 单步到程序源代码的下一行，进入函数。
stepi 单步一条机器指令，进入函数。

查看信息

backtrace、info stack、where查看函数调用栈
info program 查看程序状态
info functions 查看函数
info frame 查看栈结构
info registers 查看寄存器
